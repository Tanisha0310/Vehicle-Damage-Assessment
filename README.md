# Vehicle Damage Assessment

A computer vision project using YOLOv8 for vehicle damage detection and classification.

## Features

- Detect vehicle damage location
- Classify damage types:
  - Door Dent
  - Door Scratch
  - Bonnet Scratch
  - Windshield Crack etc (22 features)

## Tech Stack

- Python
- YOLOv8
- OpenCV
- Flask

## Training

```bash
python train.py
```

## Testing

```bash
python test.py
```
